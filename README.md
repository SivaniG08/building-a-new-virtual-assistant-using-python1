Overview:

This is a simple voice-enabled assistant built with Python. It listens to voice commands, recognizes speech, and can perform tasks such as telling the time, opening websites, locating places on maps, sending emails, etc. The assistant uses speech-to-text (STT) and text-to-speech (TTS) to interact with the user.
